# Тестирование влияния различных видов кэширования на производительность
В данном репозитории содержится четыре одинаковых API, три из которых реализуют различные виды кэширования.

Во всех проектах для реализации кэширования используются Microsoft'овские библиотеки.

Для проведения нагрузочного тестирования использовался **k6**. Скрипт тестирования находится в файле **performance-test.js**.

Для решения, использующего InMemory кэширование, проводилось три серии испытаний при размера максимального объёма кэша равным 22,000, 43,000 и 66,000, данные размеры позволяют хранить 33%, 66% и 100% обрабатываемых данных соответственно.

Для каждого решения проводилось 5 нагрузочных испытаний, а затем высчитывалось среднее значение для каждой метрики.
